Lets you work with multiple virtual desktops,
<br>
with the help of numbered buttons.
<br>
<img width="930" height="534" alt="github description" src="https://github.com/user-attachments/assets/34a30446-32f5-4691-a86d-6c158072b513" />
<br>
Features:
<br>
Numbered Buttons - Go to any virtual desktop number by pressing a button
<br>
Left and Right buttons - to switch to the left or right desktops
<br>
Add / Remove - Plus(+) and Minus(-) buttons to add or remove desktops
<br>
Back - To switch from current desktop to the previously active desktop
<br>
Minimal - Minimize the window to hide the menu and display only the buttons
<br>
Always on Top - Stay visible above other windows
<br>
Opacity - Change the transparency effect of the window
<br>
Grid Size - Change the style of button display, set the number of columns for the grid
<br>
<br>
Shortcut:
<br>
You can use the shortcut-
<br>
<strong> Ctrl + Win + "B" </strong>
<br>
to go back to the previously visited desktop.
<br>
<br>
Supported Platforms:

<br>
